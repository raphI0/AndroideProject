Esta aplicación de reloj tiene múltiples funciones relacionadas con el tiempo. Puede utilizarse como widget de reloj o como despertador. Está hecha para ayudarte a regular tu vida diaria y a dormir mejor. También puedes utilizar el cronómetro de esta aplicación para contar tu tiempo cuando corres para llevar un estilo de vida saludable o para cualquier otro propósito. Esta aplicación también puede colocarse en tu pantalla de inicio para facilitar la navegación.

Como widget de reloj puedes habilitar la visualización de las horas de otras zonas horarias, o utilizar el widget de reloj simple, pero personalizable y redimensionable. El color del texto del widget del reloj se puede personalizar, así como el color y el alfa del fondo. También puedes cambiar la forma del widget del reloj según tu elección y mostrarlo en la pantalla de inicio.

La alarma contiene todas las funciones esperadas como la selección del día, la alternancia de la vibración, la selección del tono de llamada, la repetición de la alarma o la adición de una etiqueta personalizada. Despertarse será un placer. Soporta tantas alarmas como quieras, así que no habrá más excusas para no despertarse y dormir mejor :) También admite el aumento gradual del volumen, activado por defecto. Un botón de Snooze personalizable está disponible también, por si acaso tuvieras una buena razón para usarlo. El reloj de alarma proporcionado por esta aplicación es tan simple como puede ser. Simplemente tienes que añadir cuántas veces quieres y encenderlas. Durante esto, también puede tomar la ayuda de una guía construida en esta aplicación de reloj de alarma para ayudarle a navegar a través de esta aplicación para dormir mejor. Puedes dormir mejor para que esta app te despierte a la hora establecida sin molestar tu estilo de vida. Esta alarma se puede colocar en la pantalla de inicio para facilitar el acceso a la alarma mientras puedes trabajar en otras cosas en tu dispositivo. El objetivo principal de mantener la alarma en este widget de reloj es ayudarte a programar tu tiempo de forma más efectiva. 

Con el cronómetro puedes medir fácilmente un periodo de tiempo más largo o vueltas individuales. Puedes clasificar las vueltas de diferentes maneras. También contiene vibraciones opcionales al pulsar los botones, para avisarte de que se ha pulsado el botón en caso de que no puedas mirar el dispositivo por alguna razón, o tengas prisa. Este cronómetro puede ayudarte a ponerte en forma si estás haciendo yoga o corriendo en el parque. Puedes poner el cronómetro en la pantalla de inicio para que puedas acceder fácilmente a él y modificarlo según tus necesidades sin tener que abrir el menú y encontrarlo.

Puedes configurar fácilmente un cronómetro para ser notificado de algunos eventos. También puedes cambiar su tono de llamada o activar las vibraciones. No volverás a quemar esa pizza. La cuenta atrás del temporizador también se puede pausar, no sólo detener.

Otras características adicionales incluyen, por ejemplo, evitar que el dispositivo se duerma mientras la aplicación está en primer plano o alternar entre el formato de tiempo de 12 o 24 horas. Por último, pero no menos importante, puedes decidir si la semana debe comenzar el domingo o el lunes.

Viene con material design y tema oscuro por defecto, proporciona una gran experiencia de usuario para un uso fácil.
